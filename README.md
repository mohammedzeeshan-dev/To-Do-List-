# To-Do-List
A simple and interactive To-Do List web app built using **HTML**, **CSS**, and **JavaScript**.

##  Features

- Add new tasks  
-  Delete completed tasks  
-  Data stays for the session (basic local behavior)  
-  Fully responsive design
